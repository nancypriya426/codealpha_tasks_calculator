# CodeAlpha Task 1 - Calculator Program

## Objective
Develop a simple calculator using C programming that performs basic arithmetic operations.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Switch Case Implementation

## Technologies Used
- C Programming
- GCC Compiler

## How to Run

```bash
gcc calculator.c -o calculator
./calculator
```
